# Drone-Race-TaskScheduler

Drone race game that uses random math to dictate their distance. Replicates the game of horse betting.


## Description
This program represents the use of task scheduling and how it can be used to create visual enjoyment as well as a game style of program.

## Features
- Delayed Task Scheduling
- Drone Racing, random outcomes


## Technologies
- Language / Framework: C#
- Libraries: Math


## How to run (local)
1. Clone: `git clone https://github.com/Sxoony/Drone-Race-TaskScheduler.git`
2. Navigate: `cd REPO-NAME`
3. Setup:
- Open in Visual Studio (C#)
4. Run
